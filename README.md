# FitHub - AI Powered Fitness Trainer 🏋️‍♂️

A modern fitness web application with AI-powered chatbot, voice assistant, personalized workout plans, and nutrition guidance.

## ✨ Key Features

- **AI Chatbot**: "Chitti🤖" - Personal fitness assistant powered by Gemini AI
- **Voice Assistant**: Hands-free navigation and workout guidance
- **Personalized Plans**: Custom workout routines based on goals and equipment
- **Nutrition Guide**: BMR/TDEE calculator with meal planning
- **Responsive Design**: Works seamlessly on all devices
- **Multiple Workouts**: Back, chest, arms, shoulders, legs, and full-body training

## 🚀 Quick Start

1. **Clone the repository**:

   ```bash
   git clone https://github.com/madhanmohanreddyperam06/AI-Fitness-Trainer.git
   cd AI-Fitness-Trainer
   ```

2. **Start a local server**:

   ```bash
   # Using Python
   python -m http.server 8000
   
   # Or using Node.js
   npx serve -p 8000
   ```

3. **Open in browser**: Navigate to `http://localhost:8000`

4. **Configure API Key** (optional for AI features):
   - Get API key from [Google AI Studio](https://aistudio.google.com/)
   - Update `script.js` with your API key

- **Main Dashboard**: Select workouts, access personalized plans, and use AI features

- **Voice Assistant**: Use voice commands for hands-free navigation
- **Personalized Plans**: Get custom workout routines based on your goals
